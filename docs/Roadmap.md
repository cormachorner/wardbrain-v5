# Roadmap

WardBrain is currently in pilot hardening. The near-term aim is to expand safely while keeping the deterministic gate green.

## Current Focus

- Complete Headache v1.
- Keep acute abdomen, chest pain, and breathlessness stable.
- Maintain bulk evaluation coverage across existing and new presentation blocks.
- Continue tightening negation, dynamic extraction, and red-flag specificity.

## Near-Term Priorities

- Improve admin workflows for reviewing test-case results.
- Keep guideline support coverage complete for supported diagnoses and red flags.
- Extend evaluation reports so regressions are easier to interpret.
- Continue documenting architecture, decisions, and safety constraints.

## Medium-Term Priorities

- Add more presentation blocks only after current blocks remain stable.
- Expand hostile and messy real-world case coverage.
- Improve explainability without overwhelming the UI.
- Evaluate LLM feature extraction in offline/live harnesses before any broader use.

## Later Opportunities

- Better analytics for anonymised user activity and learning gaps.
- More structured teaching feedback for students.
- Improved case authoring/import workflows.
- Potential institutional deployment hardening.

## Non-Goals For Now

- No LLM-driven diagnosis ranking.
- No real-time clinical decision support positioning.
- No copyrighted guideline bulk import.
- No uncontrolled expansion of diagnoses without tests and guideline support.
